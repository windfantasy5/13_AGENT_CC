# 调试指南 - 智能问答无响应问题

## 问题描述
提交问题后页面无响应，提交按钮禁用，页面没有显示问题和模型思考过程。

## 调试步骤

### 1. 检查后端是否正常运行

```bash

# 测试后端健康检查
curl http://127.0.0.1:8000/health

# 查看后端日志

```

### 2. 检查前端控制台

打开浏览器开发者工具（F12），查看：

**Console 标签页：**
- 是否有 JavaScript 错误
- 是否有网络请求失败
- 查找 "Failed to send message" 或其他错误信息

**Network 标签页：**
- 查找 `/api/v1/chat/message/stream` 请求
- 检查请求状态码（应该是 200）
- 查看请求响应内容
- 检查是否有 CORS 错误

### 3. 测试流式API

在浏览器控制台执行：

## 常见问题和解决方案

### 问题1：CORS错误

**症状：** 控制台显示 "Access-Control-Allow-Origin" 错误

**解决：** 检查 `backend/.env` 中的 CORS 配置

```env
CORS_ORIGINS=http://localhost:5173,http://127.0.0.1:5173
```

### 问题2：Token过期

**症状：** 返回 401 Unauthorized

**解决：** 重新登录获取新token

```javascript
// 清除旧token
localStorage.removeItem('token');
// 重新登录
```

### 问题3：对话不存在

**症状：** 返回 404 或 "会话不存在"

**解决：** 创建新对话

```javascript
// 点击"+ 新建对话"按钮
// 或在控制台执行
fetch('http://127.0.0.1:8000/api/v1/chat/conversations', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${localStorage.getItem('token')}`,
  },
  body: JSON.stringify({ title: '测试对话' }),
})
.then(response => response.json())
.then(data => console.log('New conversation:', data));
```

### 问题4：LLM服务未配置

**症状：** 后端日志显示 LLM 相关错误

**解决：** 配置 `backend/.env`

```env
# OpenAI配置
OPENAI_API_KEY=your_api_key_here
OPENAI_BASE_URL=https://api.openai.com/v1

# 或使用其他LLM服务
```

### 问题5：流式响应不支持

**症状：** 流式API返回错误，但普通API正常

**解决：** 系统会自动降级到普通API，检查是否有错误提示

## 临时解决方案

如果流式API有问题，可以临时使用普通API：

修改 `frontend/src/pages/Chat.tsx`，将 `sendMessage` 方法改为：

```typescript
const sendMessage = async () => {
  if (!inputMessage.trim() || !currentConversation || loading) return;

  const userMessage = inputMessage;
  setInputMessage('');
  setLoading(true);

  const tempUserMsg: Message = {
    id: Date.now(),
    role: 'user',
    content: userMessage,
    created_at: new Date().toISOString(),
  };
  setMessages((prev) => [...prev, tempUserMsg]);

  try {
    const response = await chatAPI.sendMessage(currentConversation.id, userMessage);
    const { user_message, assistant_message } = response.data.data;

    setMessages((prev) => [
      ...prev.filter((m) => m.id !== tempUserMsg.id),
      {
        id: user_message.id,
        role: 'user',
        content: user_message.content,
        created_at: user_message.created_at,
      },
      {
        id: assistant_message.id,
        role: 'assistant',
        content: assistant_message.content,
        tokens: assistant_message.tokens,
        model: assistant_message.model,
        created_at: assistant_message.created_at,
      },
    ]);

    await loadConversations();
  } catch (error) {
    console.error('Failed to send message:', error);
    alert('发送消息失败: ' + (error as Error).message);
    setMessages((prev) => prev.filter((m) => m.id !== tempUserMsg.id));
  } finally {
    setLoading(false);
  }
};
```
