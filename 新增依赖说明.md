# 新增依赖安装说明

## Python后端依赖

为了支持增强的PDF和网页处理功能，需要安装以下Python包：

```bash
cd backend

# PDF转Markdown（保留图片、表格描述）
pip install pymupdf4llm

# HTML转Markdown
pip install html2text

# 如果pymupdf4llm安装失败，可以单独安装PyMuPDF
pip install PyMuPDF
```

## 依赖说明

1. **pymupdf4llm**: 将PDF转换为Markdown格式，保留图片描述和表格结构
2. **html2text**: 将HTML转换为Markdown格式，用于网页内容清洗
3. **PyMuPDF (fitz)**: pymupdf4llm的底层依赖

## 可选依赖

如果需要OCR功能识别扫描版PDF：
```bash
pip install paddleocr pdf2image
```

## 验证安装

```bash
python -c "import pymupdf4llm; print('pymupdf4llm ok')"
python -c "import html2text; print('html2text ok')"
python -c "import fitz; print('PyMuPDF ok')"
```
